# ShibaSteps

Team name: mimimi

## Project Description

ShibaSteps is a goal-tracking web app that helps students build consistent habits through daily check-ins, cute Shiba-themed motivation, streaks, poop XP, and small community accountability.

## Problem

Many students set big goals but fail to maintain daily progress. Their goals often feel too large, reminders are scattered, and motivation drops quickly after the first few days.

## Solution

ShibaSteps helps users break big goals into small daily tasks. Users can check in every day, earn poop XP, maintain streaks, and share their progress with a 20-person Dog Circle.

## MVP Features

- Create a personal goal
- View daily tasks
- Complete tasks using a checklist
- Earn poop XP after completing tasks
- Track daily streaks
- Share progress to Dog Circle
- View a simple community feed

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- localStorage

## Target Users

Our target users are students who want to build better habits, stay consistent with long-term goals, and receive light social accountability from a small community.

## Future Improvements
